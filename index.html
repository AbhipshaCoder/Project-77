A console traditionally refers to a computer terminal where a user may input commands and view output such as the results of inputted commands or status messages from the computer. In PC terms, the console no longer refers strictly to the monitor but to a piece of software, such as a command prompt, where a user can input advanced commands. This type of software is called a console because it is similar in function to the hardware console, which allows users advance use of the computer system it is attached to.
